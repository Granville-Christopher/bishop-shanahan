# Bishop Joseph Shanahan Foundation

A full-stack Next.js web application built with MongoDB for dynamic content management and modern styling.

## Getting Started

1. Set up your `.env` file with your `MONGODB_URI`.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Access the admin dashboard at `http://localhost:3000/admin`.
